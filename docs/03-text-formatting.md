## Lists

### Unordered Lists

* Item 1
* Item 2
* Item 3
  * Nested item 3.1
  * Nested item 3.2
    * Double nested 3.2.1
    * Double nested 3.2.2
* Item 4

Alternative syntax:

- Item A
- Item B
- Item C

Another alternative:

+ Plus item 1
+ Plus item 2
+ Plus item 3

### Ordered Lists

1. First item
2. Second item
3. Third item
   1. Nested item 3.1
   2. Nested item 3.2
      1. Double nested 3.2.1
      2. Double nested 3.2.2
4. Fourth item

With different starting number:

5. Starting at five
6. Six
7. Seven

### Mixed Lists

1. Ordered item
   * Unordered nested
   * Another unordered
2. Another ordered
   1. Nested ordered
   2. Another nested ordered
      * Mixed with unordered
      * More unordered
3. Final ordered item

### Task Lists

- [x] Completed task
- [x] Another completed task
- [ ] Incomplete task
- [ ] Another incomplete task
  - [x] Completed subtask
  - [ ] Incomplete subtask

### Definition Lists (HTML)

<dl>
  <dt>Term 1</dt>
  <dd>Definition for term 1</dd>
  
  <dt>Term 2</dt>
  <dd>Definition for term 2</dd>
  
  <dt>Term 3</dt>
  <dd>First definition for term 3</dd>
  <dd>Second definition for term 3</dd>
</dl>
